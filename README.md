## Запустить локально:

Клонировать репозиторий

```bash
  git clone https://github.com/6ocra3/y-frontend-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Запустить проект:

```bash
  npm start
```

Запустить бэкенд-сервер:

```bash
  npm run server
```
